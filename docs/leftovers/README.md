# Leftovers

These features don't directly belong to the core focus-order functionality of the library, but they were built alongside it and have proven useful in practice. They are kept here as part of the package's history rather than as first-class API surface.

| Feature | Description |
| :-- | :-- |
| [autoFocus](./autofocus.md) | Move screen reader focus to an element on mount |
| [A11yModule.announce](./announce.md) | Reliable programmatic announcements on iOS |
