# Components

| :-- | :-- |

| [A11y.FocusTrap + A11y.FocusFrame](./A11yFocusTrap.md) | Confine screen reader focus to a subtree (modals, bottom sheets). |
| [A11y.PaneTitle + A11y.ScreenChange](./A11yPaneTitle.md) | Announce screen or panel transitions to VoiceOver and TalkBack. |
